## 4. Controls

- <kbd>Esc</kbd> or <kbd>Alt</kbd>+<kbd>F4</kbd>: quit
- <kbd>S</kbd>: start new game

### 4.1. Game

- <kbd>Enter</kbd>: hard drop
- <kbd>↑</kbd>: rotate counterclockwise
- <kbd>Space</kbd>: soft drop
- <kbd>←</kbd>: move left
- <kbd>↓</kbd>: rotate clockwise
- <kbd>→</kbd>: move right

### 4.2. Pause screen

- <kbd>P</kbd>: unpause
