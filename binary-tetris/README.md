## 3. Controls

- <kbd>Ctrl</kbd>+<kbd>R</kbd>, <kbd>Cmd</kbd>+<kbd>R</kbd>: restart
- <kbd>Alt</kbd>+<kbd>F4</kbd>, <kbd>Cmd</kbd>+<kbd>Q</kbd>: quit
- <kbd>Left</kbd>: move left
- <kbd>Down</kbd>: soft drop
- <kbd>Right</kbd>: move right
