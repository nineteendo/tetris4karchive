4k-tris

This small but rather fancy tetris was written for the 4K game programming contest. Even though it fits in as little as 4KB jar file, it features smooth animation, shadows and sound effects. It has been tested on JRE 1.4 so far.

  • Download the original submission for the competition here: 4ktris-1.0.jar.
  • Play a slightly extended version which displays a number of frames per second: 4ktris.jnlp.
  • Or download the extended version: 4ktris-FPS-1.0.jar.

The extended version would still fit into 4KB zip file if it did not contain the manifest file.

As to the jar files, you only need to run java -jar 4ktris-1.0.jar or java -jar 4ktris-FPS-1.0.jar to start them.
