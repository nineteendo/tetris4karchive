## 3. Controls

- <kbd>Alt</kbd>+<kbd>F4</kbd> or <kbd>⌘ Cmd</kbd>+<kbd>Q</kbd>: quit

### 3.1. Game

- <kbd>↑</kbd>: rotate clockwise
- <kbd>←</kbd>: move left
- <kbd>↓</kbd>: hard drop
- <kbd>→</kbd>: move right

### 3.2. Game over screen

- <kbd>Page Up</kbd>: increase level
- <kbd>Page Down</kbd>: decrease level
- <kbd>Space</kbd>: retry
