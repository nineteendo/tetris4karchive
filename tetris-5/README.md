## 3. Controls

- <kbd>Q</kbd>: rotate counterclockwise
- <kbd>W</kbd>: rotate clockwise
- <kbd>Ctrl</kbd>+<kbd>R</kbd> or <kbd>⌘ Cmd</kbd>+<kbd>R</kbd>: restart
- <kbd>Ctrl</kbd>+<kbd>W</kbd> or <kbd>⌘ Cmd</kbd>+<kbd>W</kbd>: quit
- <kbd>←</kbd>: move left
- <kbd>↓</kbd>: soft drop
- <kbd>→</kbd>: move right
