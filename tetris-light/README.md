## 3. Controls

- <kbd>Ctrl</kbd>+<kbd>R</kbd> or <kbd>⌘ Cmd</kbd>+<kbd>R</kbd>: restart
- <kbd>Ctrl</kbd>+<kbd>W</kbd> or <kbd>⌘ Cmd</kbd>+<kbd>W</kbd>: quit

### 3.1. Title screen

- <kbd>Any Key</kbd>: start game

### 3.2. Game

- <kbd>↑</kbd>: rotate clockwise
- <kbd>←</kbd>: move left
- <kbd>↓</kbd>: soft drop
- <kbd>→</kbd>: move right

### 3.3. Game over screen

- <kbd>Any Key</kbd>: retry
